# 📝 To-Do List App

🚀 A simple and interactive To-Do List application built using HTML, CSS, and JavaScript.  
This project helps users manage daily tasks efficiently while improving core frontend development skills.

---

## 📌 Overview
This project allows users to add, delete, and manage tasks dynamically using JavaScript.

It is built to strengthen key web development concepts such as:
- DOM manipulation
- Event handling
- JavaScript logic building
- Dynamic UI updates
- Local storage (optional feature if added)

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)

---

## ✨ Features
- 📝 Add new tasks
- ❌ Delete completed tasks
- ✔️ Mark tasks as completed (optional)
- 💾 Save tasks using local storage (if implemented)
- 🎨 Clean and simple UI
- 📱 Responsive design (optional)
